# **App Name**: PoshanAI: Your Pocket Dietician

## Core Features:

- User Authentication: Secure login and signup functionality to manage user profiles and data.
- Profile Creation: Users provide basic information (age, weight, height, gender, health issues, preferred diet) to personalize nutrient tracking and recommendations.
- AI-Powered Deficiency Detection Tool: LLM assesses current nutrient intake and potential deficiencies, providing personalized recommendations based on user conditions (e.g., obesity). The tool reasons over meals, nutrients and personalized recommendations.
- Dynamic Nutrient Dashboard: Visually display nutrient intake against recommended daily allowances using charts (donut, progress bars).
- Personalized Food Recommendations: Suggest food to mitigate identified nutrient deficiencies, shown in a 3D carousel format. Recommendations are tailored to the user's health conditions.
- Meal Logging with API data fetch: Allow users to input their meals, specifying food items and portion sizes. Meal data is fetched from an external API.
- Diet and Weight Tracker: Track diet and weight changes, providing personalized recommendations for workout routines and meal timing based on user's condition and goals, sending notifications as reminders.
- AI Meal Analysis: Analyze the calories and contents of a meal by taking a picture of it.

## Style Guidelines:

- Primary color: Saturated teal (#30BFBF) to convey a sense of health, balance, and trustworthiness.
- Background color: Light teal (#E0F8F8), to create a clean and calming user experience.
- Accent color: A vibrant shade of chartreuse (#BFFF00) will be used sparingly to draw attention to key elements and CTAs.
- Font pairing: 'Poppins' (sans-serif) for headlines, and 'PT Sans' (sans-serif) for body text.
- Responsive design with a 3D-themed UI, utilizing CSS Modules for styling.
- Use clear, modern icons related to nutrition and health for easy navigation and understanding.
- Subtle transitions and animations on the 3D carousel for smooth, engaging user experience.
